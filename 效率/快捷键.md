> 根据常用软件为一些通用功能定义一套相同快捷键，在使用不同软件时，据此进行相应修改。



| 功能     | Mac       | Windows         |
| ------ | --------- | --------------- |
| 上一标签   | ⌘ + [     | Alt + [         |
| 下一标签   | ⌘ + ]     | Alt + ]         |
| 关闭标签   | ⌘ + W     | Ctril + W       |
| 显示隐藏标签 | ⌥ + ⌘ + H | Ctrl + Alt + H  |
| 前进     | ⇧ + ⌘ + ] | Alt + Shift + ] |
| 后退     | ⇧ + ⌘ + [ | Alt + Shift + [ |
| 打开软件设置 | ⌘ + ,     | Alt + ,         |
| 输入法切换  | ⌥ + Space |                 |
|        |           |                 |
