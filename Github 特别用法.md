## Issues
### 管理 
+ Label 标签  
+ Milestones 里程碑  

### TaskList 语法
```
- [x] 完成部署工具的设置
- [ ] 博客文章更新
- [ ] 实现抽奖功能
```
效果如下：  
- [x] 完成部署工具的设置
- [ ] 博客文章更新
- [ ] 实现抽奖功能

### 关闭 Issue
提交时采用以下格式描述信息，在 push 后将自动关闭相应 issue：
+ fix #24
+ fixes #24
+ fixed #24
+ close #24
+ closes #24
+ closed #24
+ resolve #24
+ resolves #24
+ resolved #24
