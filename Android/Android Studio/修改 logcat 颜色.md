Settings -> Editor -> Color Scheme -> Android Logcat

logcat 的颜色默认与 console 中的预定义颜色一致，所以我们需要取消勾选 Inherit values from。

推荐的 Darcula 颜色：

```
Debug  : 6897BB
Info   : 6A8759
Warn   : BBB529
Error  : FF6B68
Assert : 9876AA
```
